Problem Statement

Write a program that reads the Student Exam Records.  The program must output an HTML table, with a row for each student who had an exam score with a grade of “A”.  An “A” is defined for this program as a score that is greater than or equal to 90. The table will be preceded by an HTML header tag identifying the table.

Following the table, two total lines must appear:

  • One line which gives the number of records found in the file that had “A” exam scores
  • One line which gives the total number of records found in the file.

The program must be broken up into reasonable functions, although no set number of functions is required. Suggested functions are documented in the following section. The controller function part03() already exists in the JavaScript file. You will need to create all the other functions.

It may help you if, prior to programming, you consult this Optional Structure for the overall structure of this program.

The program must be written so that it doesn’t know in advance how many students had an exam score of “A” nor can it know in advance who they are or their relative position within the record set.
