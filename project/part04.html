Problem Statement

Write a program that receives, via the prompt() method, five strings into five variables (one string per variable). The program obtains the length of each string and stores the length into variables. For each entered string the program will output a line like the sample output below. Then, add up each string length for a total.

Note

The sample strings shown in the sample below are just samples–the program must be able to accept any five strings keyed in and display those keyed strings).
