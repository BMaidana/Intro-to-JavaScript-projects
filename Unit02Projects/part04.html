Problem Statement

Write a program that calculates several possible tips to give to a waiter at a restaurant. Ask the user to enter the original cost of the meal and then calculate a tip at 10%, 12.5%, 15%, 17.5%, and 20%. Write the original cost of the meal to the web page. Write all the possible tips and the new final cost to the web page. Note that a meal cost may have a dollars and cents value. Do not assume that only whole numbers will be entered by the user. For full credit, the results must be displayed in three columns like the example output. Rounding of answers is not required.

Remember: You must use constants for any values in your program that do not change.
