Problem Statement

Write a program to read a Fahrenheit temperature entered by the user, convert it to Celsius and print the converted temperature. The formula for the conversion is: Celsius temperature = (Fahrenheit temperature - 32) * (5 / 9).

Note: you can hard-code the numeric values in the conversion formula.


In the following section, write the three planning lists.

Planning List

Input Variables

  • Fahrenheit temperature (number)

Output Variables

  • Fahrenheit
  • Celsius temperature (number)

Process Checklist

  1. Prompt the user to enter a Fahrenheit temperature.
  2. Convert the entered Fahrenheit temperature to Celsius using the formula: Celsius temperature = (Fahrenheit temperature - 32) * (5 / 9).
  3. Display the entered Fahrenheit temperature and the calculated Celsius temperature.

Sample Output

  Entered Fahrenheit Temperature: 77
  Calculated Celsius Temperature: 25
